Full-Stack College Project

Track student expenses with secure login, smart summaries, and budget alerts.

Student Expense Tracker is a Spring Boot, MySQL, HTML, CSS, and JavaScript project that helps students record spending, manage monthly budgets, and understand their expenses with charts and category breakdowns.

Project Highlights

Everything a student needs to manage personal spending

Secure Personal Accounts

Each user can register, log in, and manage only their own expenses through session-based authentication and BCrypt password hashing.

Expense Dashboard

Add, edit, delete, search, filter, and export expenses while viewing daily totals, monthly totals, and category summaries.

Visual Insights

Pie and bar charts powered by Chart.js help students quickly understand where money is going across categories and months.

Budget Tracking

Students can set a monthly budget and get alerts when they cross the limit so they can control spending early.

Tech Stack

Built with a beginner-friendly full-stack setup

Run Locally

Quick start on your computer

  1. Clone the repository from GitHub.
  2. Configure MySQL credentials in src/main/resources/application.properties.
  3. Run database/schema.sql and optionally database/sample-data.sql.
  4. Start the backend with mvn spring-boot:run.
  5. Open http://localhost:8080/login.html in your browser.
